# Directory structure

Examples of directory structures for documentation purposes.

```
.
├── folder/
│   ├── folder/
│   └── file.ext
├── folder/
│   ├── folder/
│   │   ├── file.ext
│   │   └── file.ext
│   ├── file.ext
│   └── file.ext
├── folder/
│   └── file.ext
└── file.ext
```

```
.
├── folder/
│   ├── file.ext
│   ├── file.ext
│   └── folder/
│       ├── folder/
│       │   └── file.ext
│       └── file.ext
├── file.ext
└── folder/
    └── folder/
        ├── file.ext
        └── file.ext
```
