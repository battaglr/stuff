# Comments in Sass

## Documentation comments

```scss
////
// Add `border-box` property to all elements,
// including pseudo-elements.
//

*,
*::before,
*::after {
  box-sizing: border-box;
}
```

## Section comments

```scss
// > First level
// ---------------------------------------------------------------- //

// >> Second level
// ------------------------------------------------------ //

// >>> Third level
// -------------------------------------------- //

// >>>> Fourth level
// ---------------------------------- //
```
