# Clearfix

A simpler *clearfix* by [Thierry Koblentz](http://cssmojo.com/latest_new_clearfix_so_far/).

```css
.Cf::after {
  content: '';
  clear: both;
  display: table;
}
```

> [`pseudo-elements` support on Can I Use](http://caniuse.com/#feat=css-gencontent)  
> *IE8 only supports the single-colon syntax.*
